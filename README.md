# ParkEase

A **multi-user parking management system** designed to streamline reservation and administration workflows. Built with **Flask (backend)** and **Vue.js (frontend)**, the platform provides **real-time spot allocation**, **role-based access control (RBAC)**, and a **responsive dashboard** for users and administrators.
